Pychron Public Database

API

Pychron offers a public api for accessing the database. API

Example Uses

  • Limit

    Get first two analyses

    Using Browser
    /api/v0/analyses?q={"limit":2}

    Using Python

    {% highlight 'python' %} import requests import json url = 'http://<host>/api/v0/analyses' headers = {'Content-Type': 'application/json'} params = dict(q=json.dumps(dict(limit=2))) response = requests.get(url, params=params, headers=headers) assert response.status_code == 200 print response.json() {% endhighlight %}

  • Filter

    Get analysis with "id" equal to 3

    Using Browser
    /api/v0/analyses?q={"filters":[{"name":"id","op":"eq","val":3}]}

    Using Python

    {% highlight 'python' %} import requests import json url = 'http://<host>/api/v0/analyses' headers = {'Content-Type': 'application/json'} filters = [{'name':'id', 'op':'eq', 'val': 3}] params = dict(q=json.dumps(dict(filters=filters))) response = requests.get(url, params=params, headers=headers) assert response.status_code == 200 print response.json() {% endhighlight %}

  • Filter by Relation

    Get analyses with "identifier" equal to "23966"

    Using Browser
    /api/v0/analyses?q={"filters":[{"name":"iposition__identifier","op":"has","val":"23966"}]}

    Using Python

    {% highlight 'python' %} import requests import json url = 'http://<host>/api/v0/analyses' headers = {'Content-Type': 'application/json'} filters = [{'name':'iposition_identifier', 'op':'has', 'val': '23966'}] params = dict(q=json.dumps(dict(filters=filters))) response = requests.get(url, params=params, headers=headers) assert response.status_code == 200 print response.json() {% endhighlight %}
